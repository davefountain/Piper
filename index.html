Piper

Piper is a thin layer over Javascript. It removes brackets { } and uses significant whitespace to implement blocks.

Rules:

  • Blocks are identified by indenting with 4 spaces per level
  • No tabs allowed for indenting - only spcaes
  • The only place you can declare a variable is within a global or local block - ideally at the beginning of the script, function or method
  • LET, CONST and VAR are not permitted
  • Omit the ( ) brackets in FOR and IF statements
  • In a FOR statement Piper will add a LET for you after the (, so
    for i = 0; i <= pointCount; i += 1
    becomes
    for (let i = 0; i <= pointCount; i += 1) {
